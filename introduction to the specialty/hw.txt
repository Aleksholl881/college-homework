1.Данные — это любая информация, представленная в формализованном виде, пригодном для хранения, передачи и обработки компьютером.


2.Виды данных и их пример:
Текстовые- "Привет, мир!"
Числовые- 42, -15
Логические- true, false
Дата и время- 2024-01-15, 14:30:00
Бинарные- Фотография image.jpg, песня song.mp3

3.Тип данных — это характеристика, которая определяет:

1.Множество допустимых значений для данных этого типа.
2.Множество допустимых операций, которые можно выполнять над этими значениями.
3.Способ хранения данных в памяти компьютера.

Почему тип данных важен при программировании?
Эффективное использование памяти. Компьютер выделяет разный объем памяти для разных типов. Например, для хранения небольшого целого числа (тип byte) требуется 1 байт, а для вещественного числа с большой точностью (тип double) — 8 байт. Использование правильного типа экономит ресурсы.

Контроль корректности операций. Тип данных не позволяет выполнять бессмысленные операции. Компилятор или интерпретатор сообщит об ошибке, если вы попытаетесь, например, умножить строку на дату. Это помогает находить ошибки на ранних этапах.

Повышение читаемости кода. Явное указание типов делает код более понятным для других программистов и для вас самих в будущем. Легко понять, что переменная userAge предназначена для целых чисел, а userName — для текста.

Обеспечение правильности вычислений. Разные типы данных по-разному обрабатываются в арифметических операциях. Деление целых чисел (5 / 2) может дать результат 2 (целочисленное деление), в то время как деление чисел с плавающей точкой (5.0 / 2.0) даст точный результат 2.5.

4.Основные типовые данные:
Целые числа,Числа с плавающей точкой, Строки, Булевы значения, Символы.
